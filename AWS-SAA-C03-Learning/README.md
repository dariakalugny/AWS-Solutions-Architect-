# AWS Certified Solutions Architect - Associate Learning Environment

This workspace is a structured study environment for the **AWS Certified Solutions Architect - Associate (SAA-C03)** certification.

It is organized around the official exam domains:

| Domain | Weight |
| --- | ---: |
| Design Secure Architectures | 30% |
| Design Resilient Architectures | 26% |
| Design High-Performing Architectures | 24% |
| Design Cost-Optimized Architectures | 20% |

## How To Use This Workspace

1. Start with `00-exam-overview`.
2. Study one domain at a time.
3. For every AWS service, fill in the service template:
   - What problem does it solve?
   - When should I use it?
   - When should I avoid it?
   - What are the common exam traps?
4. Complete the hands-on labs.
5. Answer practice questions and write down why wrong answers are wrong.
6. Review architecture patterns before mock exams.

## Main Goal

By the end of this learning path, you should be able to look at a scenario and choose the best AWS architecture based on:

- Security
- Resilience
- Performance
- Cost
- Operational fit

The exam is less about memorizing service names and more about choosing the best design for a specific business requirement.
