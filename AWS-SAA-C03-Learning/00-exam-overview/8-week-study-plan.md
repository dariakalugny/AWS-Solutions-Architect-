# 8 Week Study Plan

## Week 1: Foundations

- AWS global infrastructure: Regions, Availability Zones, edge locations
- Shared responsibility model
- IAM basics: users, groups, roles, policies
- VPC basics: subnets, route tables, internet gateway, NAT gateway
- Start flashcards for core services

## Week 2: Security

- IAM, IAM Identity Center, STS, Organizations, SCPs
- KMS, ACM, Secrets Manager, Parameter Store
- Security Groups vs Network ACLs
- WAF, Shield, GuardDuty, Inspector, Macie, Security Hub
- Lab: secure VPC with public/private subnets

## Week 3: Compute and Load Balancing

- EC2, Auto Scaling, ELB
- Lambda, Fargate, Elastic Beanstalk
- ECS vs EKS vs Lambda vs EC2
- Instance families and purchasing options
- Lab: highly available web app architecture

## Week 4: Storage

- S3, S3 Glacier, lifecycle policies, replication
- EBS, EFS, FSx
- AWS Backup
- Storage Gateway, DataSync, Transfer Family
- Lab: S3 lifecycle and versioning design

## Week 5: Databases

- RDS, Aurora, Aurora Serverless
- DynamoDB, ElastiCache
- Redshift, DocumentDB, Neptune, Keyspaces, QLDB
- Read replicas, Multi-AZ, backups, global tables
- Lab: choose a database for five scenarios

## Week 6: Networking and Resilience

- Route 53 routing policies
- CloudFront, Global Accelerator
- VPC peering, Transit Gateway, PrivateLink
- Site-to-Site VPN, Client VPN, Direct Connect
- DR strategies: backup and restore, pilot light, warm standby, active-active

## Week 7: Analytics, Integration, and Cost

- SQS, SNS, EventBridge, Step Functions, MQ
- Athena, Glue, Kinesis, EMR, Redshift, QuickSight, OpenSearch
- Cost Explorer, Budgets, Savings Plans, Trusted Advisor
- Cost tradeoffs: NAT Gateway, data transfer, storage classes, Spot

## Week 8: Review and Exam Practice

- Review all service cheatsheets
- Do 2 full practice exams
- Track every wrong answer
- Re-study weak domains
- Build 3 architecture diagrams from memory
- Final review: IAM, VPC, S3, RDS, DynamoDB, Route 53, ELB, Auto Scaling, CloudFront
